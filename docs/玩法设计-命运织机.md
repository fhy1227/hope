## 核心创意

每波战斗结束后，除了商店之外，玩家还可以从 3 张"命运卡牌"中选择一张。卡牌效果是本局内生效的全局加成，且特定卡牌组合会触发连锁效果。

### 三选一规则

```
[波次结束]
    │
    ▼
[命运织机阶段] ← 新增，在商店之前
    ├── 抽 3 张卡（从所有可用卡牌中按权重抽取）
    ├── 显示卡牌名称、描述、稀有度（颜色区分）
    ├── 显示已持有卡牌 → 可触发连锁预览
    ├── 玩家选择 1 张 → 生效并持有
    └── 未选择的卡牌消失
    │
    ▼
[商店阶段] ← 原有
```

### UNIQUE 机制

- 每张卡牌**每局只能选择一次**
- 连锁触发后**持续生效**
- 卡牌效果**不可逆转**（选中即绑定）
- 史诗卡出现时伴随**特殊光效和音效**

## 系统流程

### 命运织机出现时机

| 触发条件 | 说明 |
|----------|------|
| 每波战斗结束 | 紧接在 WaveCompleted 之后 |
| 仅在 RunPhase.Shop 之前 | 先选卡牌，再进商店 |
| 第 1 波结束后必出 | 引导玩家了解系统 |
| 之后每波固定出现 | 每波都有一次选择机会 |

### 卡牌出现的波次进度

```
波次 1 ─ 3 张基础卡（新手引导）
波次 2 ─ 3 张基础卡
波次 3 ─ 2 基础 + 1 稀有
波次 4 ─ 1 基础 + 2 稀有
波次 5+ ─ 混合出牌（含史诗概率）
```

随着波次增加，稀有卡和史诗卡的出现概率逐步提升。

### 已持有卡牌查看

在 HUD 上增加快捷入口，打开后以网格形式展示所有已持有卡牌，卡片左侧显示连锁状态提示。

### 连锁激活规则

```
1. 玩家获得一张卡牌后，检查所有已持有卡牌
2. 若满足某个连锁的所有卡牌条件 → 连锁激活
3. 连锁效果立即生效（被动型）或注册到对应系统
4. 连锁激活时有视觉反馈：
   - 连锁卡牌边框变为紫色
   - 屏幕中央显示"连锁激活！XXX"
   - 效果简述
5. 连锁效果可与单卡效果叠加
6. 同一张卡牌可参与多个连锁
```

## 卡牌出现规则

### 权重计算

```
卡牌出现权重 = 基础权重 × 波次系数 × 已持有系数

基础权重:
  Common = 60
  Rare   = 30
  Epic   = 10

波次系数:
  波次 1-2: Common ×1.0, Rare ×0.0, Epic ×0.0
  波次 3-4: Common ×1.0, Rare ×0.5, Epic ×0.0
  波次 5-6: Common ×1.0, Rare ×0.8, Epic ×0.3
  波次 7+:  Common ×1.0, Rare ×1.0, Epic ×0.5

已持有系数:
  已经持有的卡牌权重 ×0（不重复出现）
```

### 三张卡牌抽取算法

```csharp
public List<CardConfig> DrawThreeCards(int currentWave, List<int> ownedCardIds)
{
    var pool = GetAllAvailableCards()
        .Where(c => !ownedCardIds.Contains(c.CardId))
        .Select(c => new
        {
            Card = c,
            Weight = CalculateWeight(c, currentWave)
        })
        .ToList();

    var result = new List<CardConfig>();
    for (int i = 0; i < 3; i++)
    {
        var drawn = WeightedRandom(pool);
        result.Add(drawn.Card);
        pool.Remove(drawn); // 移除已抽取的，防止三张重复
    }

    // 保底：如果玩家连续 3 波没见到稀有卡，下一波必出至少一张稀有
    return result;
}
```

### 保底机制

| 保底条件 | 效果 |
|----------|------|
| 连续 3 波未出现稀有卡 | 下一波至少 1 张稀有 |
| 连续 6 波未出现史诗卡 | 下一波至少 1 张史诗 |
| 已持有卡牌数 < 3 | 优先出现可连锁的卡牌 |

### CardEffectResolver 效果路由

```csharp
public static class CardEffectResolver
{
    // 卡牌效果类型 → 对应的系统调用
    public static void ApplyEffect(CardConfig card, RunStats stats)
    {
        switch (card.EffectType)
        {
            // ——— 属性类：直接修改 RunStats ———
            case "stat_max_hp":
                stats.MaxHealth += (int)card.EffectValue;
                break;
            case "stat_damage":
                stats.Damage *= (1 + card.EffectValue / 100f);
                break;
            case "stat_attack_speed":
                stats.AttackSpeed *= (1 + card.EffectValue / 100f);
                break;
            // ... 以此类推

            // ——— 触发类：注册到 EventBus ———
            case "on_kill_heal_pct":
                // 注册到 EventBus.EnemyKilled
                EventBus.Instance.EnemyKilled += (enemy) => {
                    HealPlayer(GetEffectValue("on_kill_heal_pct"));
                };
                break;
            case "on_hit_bounce":
                // 注册到 EventBus.DamageDealt
                EventBus.Instance.DamageDealt += (dmg) => {
                    if (Random() < 0.2f) BounceToNearbyEnemy(dmg * 0.5f);
                };
                break;
            // ... 以此类推

            // ——— 技能类：修改技能参数 ———
            case "skill_roll_blink":
                // 替换 RollAction 的移动方式
                RollAction.SetMovementMode(MovementMode.Blink);
                break;
        }
    }
}
```

```csharp
// RunManager.cs — 在 WaveCompleted 和 ShowShop 之间插入
private void OnWaveCompleted()
{
    // 原有逻辑
    StopSpawning();

    // 新增：命运织机阶段
    if (FateCardManager.Instance.IsEnabled)
    {
        SetPhase(RunPhase.FateCard); // 新增阶段
        // 打开命运织机 UI
        // 等待玩家选择完成后再进入商店
        FateCardManager.Instance.OnCardSelected += OnCardSelected;
        FateCardPanel.Instance.Show(FateCardManager.Instance.DrawCards(CurrentWave));
        return;
    }

    // 无命运织机 → 直接进商店
    SetPhase(RunPhase.Shop);
    ShowShop();
}

private void OnCardSelected(CardConfig card)
{
    FateCardManager.Instance.SelectCard(card);
    FateCardManager.Instance.OnCardSelected -= OnCardSelected;

    // 选择完成后进入商店
    SetPhase(RunPhase.Shop);
    ShowShop();
}
```

```csharp
// 连击卡 (R03) 集成 — 在 WeaponSlot 中
public class WeaponSlot : Node2D
{
    private int _comboCount;
    private Enemy _currentTarget;

    public void OnHitEnemy(Enemy enemy)
    {
        float comboBonus = FateCardManager.Instance.GetEffectValue("on_hit_combo");
        if (comboBonus > 0)
        {
            if (enemy != _currentTarget)
            {
                _comboCount = 0;
                _currentTarget = enemy;
            }
            _comboCount++;
            float bonus = Mathf.Min(_comboCount * 0.05f, 0.5f);
            _currentDamage *= (1 + bonus);
        }
    }
}
```

```csharp
// 弹射卡 (R02) 集成 — 在 Projectile 中
public class Projectile : Node2D
{
    private void OnHitEnemy(Enemy enemy)
    {
        float bounceChance = FateCardManager.Instance.GetEffectValue("on_hit_bounce");
        if (bounceChance > 0 && GD.Randf() < bounceChance)
        {
            var nearby = FindNearbyEnemy(enemy, 100f);
            if (nearby != null)
            {
                var bounceDmg = _damage * 0.5f;
                nearby.TakeDamage(bounceDmg);
            }
        }
    }
```
