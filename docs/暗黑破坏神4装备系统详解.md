## 1. 装备槽位与武器类型

| 类别 | 槽位 | 数量 |

## 2. 物品品质体系

### 2.1 品质等级一览

| 品质 | 颜色 | 词缀数 | 可淬炼 | 可精工 | 备注 |
|------|:----:|:------:|:------:|:------:|------|
| **普通 (Common)** | 白 | 0 | 否 | 否 | 无任何加成 |
| **魔法 (Magic)** | 蓝 | 1 | 否 | 否 | 过渡用 |
| **稀有 (Rare)** | 黄 | 3 | 是 | 是 | 可升级为传奇 |
| **传奇 (Legendary)** | 橙 | 4 + 1 威能 | 是 | 是 | 核心玩法 |
| **独特 (Unique)** | 绿/金 | 4 固定 + 1 独特威能 | 否 | 是 | 固定词缀 |
| **神话独特 (Mythic Unique)** | 紫/红 | 4 固定 + 1 神话威能 | 否 | 是 | 最稀有 |

### 2.2 子品质：普通 vs 远古

| 子品质 | 最大物品强度 | 强效词缀 | 掉落条件 |
|--------|:----------:|:--------:|----------|
| **普通 (Normal)** | 750 | 无 | 任何难度 |
| **远古 (Ancestral)** | 800 | 至少 1 条 | 苦痛 1 (Torment 1) 及以上 |

> 远古传奇掉落时**必定带至少 1 条强效词缀**，最多可达 4 条。
>
> 远古独特同样至少带 1 条强效词缀。

### 2.3 升级路上的品质变化

| 等级阶段 | 掉落趋势 |
|----------|----------|
| 1-27 | 普通和魔法为主 |
| 28-49 | 80% 非传奇为稀有 (黄装) |
| 50-59 | 传奇掉落率逐渐提升 |
| 60 (满级) | 传奇为主，解锁苦痛难度 |
| 苦痛 1+ | 远古装备开始掉落 |

## 3. 物品强度 (Item Power)

### 3.1 基本概念

物品强度是衡量装备基础数值的**核心指标**，直接影响词缀的数值范围：

```
物品强度越高 → 词缀数值范围越大 → 装备越强
```

### 3.2 强度档位

| 阶段 | 物品强度范围 | 说明 |
|:----:|:----------:|------|
| 升级 | 0-750 | 随角色等级提升 |
| 满级非远古 | 750 | 传奇上限 750 |
| 苦痛 1 远古 | 800 | 远古装备起始 |
| 高等级怪掉落 | 850 | 接近 70 级怪物必掉 850 |
| 精工提升 | 850+ | 通过精工突破上限 |

### 3.3 强度 vs 词缀的选择

> **武器优先看物品强度** —— 武器伤害是技能倍率的基数，强度的重要性远超词缀。
>
> **防具和首饰** —— 在强度达到一定阈值后，词缀搭配比强度更关键。

## 4. 固有属性 (Inherent Stats)

每件高于普通品质的装备都有**固有物理特性**，是固定属性，**无法通过附魔更改**，但数值会随物品强度提升。

### 4.1 护甲与首饰固有属性

| 装备位 | 固有属性 |
|--------|----------|
| 护符 | +X% 全元素抗性 |
| 戒指 | +X% 全元素抗性、+X% 特定元素抗性 |
| 头盔/胸甲/手套/裤子 | 基础护甲 |
| 靴子 | 基础护甲 + 以下之一：最大闪避次数 / 闪避加速 / 闪避冷却缩减 |

### 4.2 独特物品的特殊固有属性

独特物品的固有属性可能与常规不同，例如 `Ancient's Oath`（先祖之誓）提供易伤伤害而非持续性伤害。

## 6. 强效词缀 (Greater Affixes)

### 6.1 基本规则

- 仅出现在**远古**品质的装备上
- **至少 1 条**，最多可 4 条全为强效
- 数值为普通词缀最大值的 **1.5 倍**
- 掉落时有**特殊音效**和**图标提示**

### 6.2 重要注意事项

> ⚠️ **绝对不要附魔强效词缀！**
>
> 在附魔（Enchant）窗口重铸强效词缀会将其**替换为普通词缀**，无法恢复。

### 6.3 获取途径

| 来源 | 说明 |
|------|------|
| 苦痛 1+ 所有掉落 | 远古传奇/独特必带强效 |
| 赌博 (Obols) | 60 级后有几率出强效 |
| 制作神话独特 | 随机获得强效词缀 |

## 7. 传奇威能 (Legendary Aspects)

### 7.1 基础机制

传奇威能是 D4 装备系统的**核心自定义机制**。每件传奇物品包含一个威能，可分解收录进**威能法典**，之后无限次使用。

```
获取威能 → 分解传奇 → 收录法典 → 铭印到目标装备
```

### 7.2 威能分类与装备位

| 威能类型 | 可铭印的装备位 |
|----------|---------------|
| **进攻 (Offensive)** | 1H武器、2H武器、手套、戒指、护符 |
| **防御 (Defensive)** | 头盔、胸甲、裤子、盾牌、护符 |
| **资源 (Resource)** | 戒指、护符 |
| **通用 (Utility)** | 盾牌、头盔、胸甲、裤子、靴子、手套、护符 |
| **移动 (Mobility)** | 靴子、护符 |

### 7.3 威能强度加成

| 装备位 | 威能强度倍率 |
|--------|:----------:|
| 护符 | ×1.5 (所有类型) |
| 双手武器 | ×2.0 (仅进攻) |
| 其他 | ×1.0 |

### 7.4 威能获取来源

| 来源 | 说明 |
|------|------|
| 地下城 | 首次通关固定奖励（基础等级） |
| 掉落分解 | 可升级法典中对应威能的等级 |
| 赌博 | 随机出传奇装备 |
| Boss 掉落 | 高几率出高品质威能 |

> 威能法典中的威能是**永久**的。分解更高数值的同类威能会自动升级法典。

## 8. 独特物品 (Unique & Mythic Unique)

### 8.1 独特物品 (Unique)

- 词缀**固定不可更改**（不能附魔、不能淬炼）
- 独特威能是**专属的、不可提取**的
- 远古独特至少带 1 条强效词缀
- 可通过 Boss 战**定向刷取**

### 8.2 神话独特 (Mythic Unique)

- 比独特更稀有，是最顶级的装备
- 可在苦痛 1+ 的任何来源掉落
- **所有 Boss 战均有更高的掉落率**
- 可通过**制作**获得（2 个辉光火花 + 符文）

### 8.3 神话独特制作

| 方式 | 材料 | 结果 |
|------|------|------|
| 铁匠随机制作 | 2 辉光火花 | 随机一件神话独特 |
| 珠宝匠定向制作 | 2 辉光火花 + 特定符文 | 指定神话独特 |
| 赫拉迪姆方块升级 | 5 混沌碎片 | 独特 → 随机同部位神话 |

获取辉光火花的方式：
- 分解神话独特
- 完成赛季旅程奖励

## 9. 淬炼 (Tempering)

### 9.1 基本规则

- 在铁匠处对**稀有或传奇**装备进行
- 为装备添加 **2 个淬炼词缀**
- **独特物品不可淬炼**
- 淬炼手册通过各类内容掉落

### 9.2 淬炼限制

```
每件装备只能淬炼有限次数（淬炼耐久度）
超过后无法再修改
必须至少有 2 个不同的淬炼词缀才能开始精工
```

### 9.3 淬炼流程

```
收集淬炼手册 → 选择手册 → 随机获得淬炼词缀 → 确认或重铸(有限次)
```

|:------:|:----------:|

### 10.2 精工提升效果

| 等级 | 效果 |
|:---:|------|
| 1-3 | 所有词缀数值 +5% |
| **4** | **随机一个词缀 +25%（变蓝）** |
| 5-7 | 所有词缀数值 +5% |
| **8** | **随机一个词缀 +25%（两个 +25% 变黄）** |

| **12** | **随机一个词缀 +25%（三个 +25% 变橙）** |

### 10.3 精工重置

> 可**重置**精工等级（回到 0 级），消耗材料，不消耗装备。

### 10.4 精工材料

## 12. 附魔 (Enchanting)

### 12.1 机制

在**神秘学家 (Occultist)** 处，可以重铸稀有或传奇装备上的一条词缀：

```
选中一条词缀 → 随机替换为同词缀池中的另一条 → 消耗金币和材料
可无限次重铸(费用递增)
```

### 12.2 限制

- **固有属性无法附魔**
- **强效词缀不可附魔**（会降级为普通）
- **独特物品不可附魔**
- 每件装备**只能重铸一条词缀**

## 15. 赌博系统 (Purveyor of Curiosities)

### 15.1 基本机制

使用**低语古币 (Obols)** 进行赌博，换取随机装备：

| 赌博目标 | 古币消耗 | 品质范围 |
|----------|:-------:|----------|
| 武器 | 75 | 普通 → 神话独特 |
| 护甲 | 55 | 同上 |
| 首饰 | 45 | 同上 |
| 随机 | 35 | 同上 |

### 15.2 重要提示

- 60 级后有几率出**强效词缀**
- 适合**补全特定装备位空缺**
- 独特/神话独特几率极低，但存在可能
- 远古赌博结果受当前角色等级影响

## 16. 威能法典 (Codex of Power)

### 16.1 法典机制

```
分解传奇 → 威能录入法典 → 可无限次铭印
铭印消耗金币和材料 → 目标装备获得该威能
```

| 特性 | 说明 |
|------|------|
| 账户共享 | 同一账号下所有角色共用 |
| 自动升级 | 分解更高数值威能自动升级 |
| 赛季重置 | 赛季角色每赛季重置，永恒角色永久 |
| 无限使用 | 录入后可不限次数铭印 |

### 16.2 铭印效果

| 目标装备 | 结果 |
|----------|------|
| 稀有 → 铭印 | 变为传奇，保留原有词缀 |
| 传奇 → 铭印 | 覆盖原有威能 |

## 17. 幻化系统 (Transmog/Wardrobe)

- 分解装备即可**永久解锁**其外观
- 在衣橱中可**任意切换**已解锁的外观
- 幻化**不影响属性**，纯外观自定义
- 支持整体配色方案调整

## 18. 与暗黑2对比分析

### 18.1 核心差异

| 维度 | 暗黑破坏神 II | 暗黑破坏神 IV |
|------|-------------|-------------|
| **系统名称** | 财宝阶层 (TC) 递归 | 模块化装备 + 威能系统 |
| **核心随机性** | 类型、品质、词缀层层独立掷骰 | 掉落后通过威能/淬炼自定义 |
| **物品类型** | 严格的 TC 分级限制掉落 | 软性类型限制，偏向性掉落 |
| **品质降级** | Unique 失败降 3 倍耐久 Rare | 无降级机制，直接决定品质 |
| **词缀** | 随 Alvl 完全随机，可极低概率出极品 | 词缀池按装备位划分，定向性强 |
| **MF 寻宝** | 线性递减（有具体公式） | 直接影响品质，但受难度影响更大 |
| **保底机制** | 几乎无保底 | 威能法典保底、制作保底、Boss 定向刷 |
| **孔洞系统** | 天然孔 + Larzuk 打孔 | 宝石孔 + 散射棱镜加孔 |
| **自定义程度** | 极低（成品属性固定） | 极高（威能+淬炼+精工自由组合） |
| **装备交易** | 完全自由交易 | 可交易（未加工过的传奇/稀有） |

### 18.2 暗黑4继承的核心设计理念

1. **递归掉落树** —— TC 系统的核心理念被延续，但实现为更简化的偏向性掉落
2. **多难度渐进** —— 苦痛 1-4 对标 D2 的噩梦/地狱难度分层
3. **Endgame Boss 系统** —— 类似 D2 的关底 Boss 掉落特定独特物品
4. **物品强度体系** —— 继承 D2 的 Ilvl/Qlvl 概念，但简化为一维数值
5. **MF 仍未完全消失** —— 虽然不再有 MF 属性，但难度直接影响品质

### 18.3 暗黑4的重大创新

| 创新 | 解决的核心问题 |
|------|---------------|
| **威能法典** | 装备"毕业即报废"，好威能可以重复利用 |
| **淬炼 + 精工** | 装备掉落只是起点，后续养成空间大 |
| **定向 Boss 刷取** | 减少 D2 那种"刷了 1000 次不出"的挫败感 |
| **强效词缀** | 给完美的装备更多追求空间 |
| **模块化词缀** | 装备位决定词缀池，不再是全程随机 |
